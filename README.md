## Upload to GitHub — exact steps

### Step 1 — Delete everything in your current repo
Go to your repo → Settings → Danger Zone → Delete repository.
Create a new repo named `echo11-frequencies`.

### Step 2 — Upload the astronaut image
In the new repo root, upload your astronaut image.
**Rename it exactly: `astronaut.jpg`**
This is what the splash screen loads.

### Step 3 — Upload the src folder
Click `Add file → Upload files`
Open the ZIP, then open the `echo11-final` folder.
Drag in EVERYTHING you see:
```
src/              ← drag the whole folder
index.html
package.json
vite.config.js
```
Commit.

### Step 4 — Connect to Vercel
Go to vercel.com → New Project → Import from GitHub
Select echo11-frequencies → Deploy
Vercel detects Vite automatically. No config needed.
